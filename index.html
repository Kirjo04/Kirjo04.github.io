What is a URL?

Introduction

Does the word, Uniform Resource Locator sound familiar? Most likely, it doesn't. But the acronym URL probably is. But what is a URL exactly? The URL is one of 5 technology components of the web. The web has 5 aspects to it: presentation, client, web server, addressing, and communication protocols. Each of these has its own unique component, the URL is the component for addressing.

Key Terms

  • Hypertext Transfer Protocol (HTTP) - The protocol browsers use to transfer data
  • Uniform Resource Locator (URL) - a subset of URIs, identifies a resource and provides a means of locating that resource
  • Domain Name System (DNS) - The server that stores all domains.
  • Top-Level Domain - The final segment of the domain (e.g. .com, .org, etc.)
  • Path - The portion of a URL that specifies the exact location of a resource within a website.
  • Search Engine Optimization (SEO) - improving the visibility of your site so it appears in search engines
  • Uniform Resource Identifier (URI) - a compact sequence of characters that identifies an abstract or physical resource
