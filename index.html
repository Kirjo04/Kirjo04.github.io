Introduction

Does the word, Uniform Resource Locator sound familiar? Most likely, it doesn't. But the acronym URL probably is. But what is a URL exactly? The URL is one of 5 technology components of the web. The web has 5 aspects to it: presentation, client, web server, addressing, and communication protocols. Each of these has its own unique component, the URL is the component for addressing.

Key Terms

  1. Hypertext Transfer Protocol (HTTP) - The protocol browsers use to transfer data
  2. Uniform Resource Locator (URL) - The code that allows browsers to access a resource
  3. Domain Name System (DNS) - The server that stores all domains.
