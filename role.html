What is a URL?

Role of URLs

The role of a URL (Uniform Resource Locator) is to provide the specific address used to locate and access a resource on the internet.

Purpose of URLs

The purpose of a URL is to allow users and browsers to find and retrieve a specific webpage or resource on the internet.

1. URL is inputted into the browser by the user
2. Browser sends URL to DNS
3. DNS finds the IP address that corresponds to the URL
4. DNS sends IP address back to browser
5. Browser sends request to the server the IP address points to
6. Server sends the content back
7. Browser displays the information
