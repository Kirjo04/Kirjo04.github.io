What is a URL?

Types of URLs

Absolute A full address for a website
Relative
  • Uses the path relative to the current page, without including the full domain.
  • Used to map to another resource on the current website
Callback Call a user back to a website after completing an action on a third party site
Vanity A shorter version of a long URL
Canonical Tell a browser which URL is correct when multiple URLs point to the same website
Deep Links directly to a specific resource
Protocol-Relative Uses the same protocol as the current webpage
